# HTWecker

This project is based on:

* [Webkid Riot Starterkit](https://github.com/wbkd/riotjs-startkit)

## Installation

```
$ npm install
```

You should additionally build `semantic-ui` using [gulp](http://gulpjs.com/):

```
cd src/semantic
gulp build
```

## Development

```
$ npm run dev
```

Now the server is runnning on localhost:1337


## Build

```
$ npm run build
```
